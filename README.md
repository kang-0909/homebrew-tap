# homebrew-tap

Homebrew tap for RouteTone.

## Install

```bash
brew tap kang-0909/tap
brew install --cask routetone
```

## Note

The `routetone` cask expects GitHub releases from:

```text
https://github.com/kang-0909/RouteTone
```

After each new RouteTone release, update the cask `version` and `sha256`.
